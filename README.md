WazePortalBR
WazePortalBR é um projeto criado para auxiliar concessionárias e prefeituras no monitoramento de dados provenientes do programa Waze for Cities.

Sobre o Projeto
O objetivo é fornecer uma plataforma eficiente e amigável para a análise e acompanhamento de informações de trânsito, contribuindo para a melhoria da mobilidade urbana e gestão viária.

Tecnologias Utilizadas
PHP 8 para o backend, garantindo alta performance e suporte a recursos modernos.
Twig como motor de templates, oferecendo um sistema flexível e seguro para a renderização de páginas.
Status do Projeto
O projeto está em estágio inicial de desenvolvimento e atualmente não está disponível online.

Colaboração e Sugestões
Estamos abertos à colaboração de membros da comunidade Waze BR e de desenvolvedores interessados em contribuir para o aprimoramento do sistema. Feedbacks e sugestões são muito bem-vindos!
